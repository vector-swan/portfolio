# Push to GitHub - Instructions

## After creating the GitHub repository:

Replace `YOUR_USERNAME` with your GitHub username in the commands below, then run:

```bash
# Add the remote repository
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git

# Push to GitHub
git push -u origin main
```

## If you haven't created the repo yet:

1. Go to: https://github.com/new
2. Repository name: `portfolio`
3. Leave all checkboxes UNCHECKED (don't add README, .gitignore, or license)
4. Click "Create repository"
5. Then run the commands above
